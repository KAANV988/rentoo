Rentoo - A Full-Stack Property Rental Platform

Welcome to Rentoo, a complete web application designed to streamline the process of finding and listing rental properties. This platform provides a secure, intuitive, and efficient experience for both renters and property owners.

✨ Features

Rentoo is a feature-rich application built with a professional, multi-page architecture.

For Everyone:

Advanced Search: A powerful search bar with advanced filters for price, bedrooms, and bathrooms.

Dynamic Property Listings: A beautiful, modern UI that displays property listings fetched from the backend.

Detailed Property View: A dedicated page for each property, featuring an image gallery, detailed description, and owner contact information.

Interactive Map Links: Each property includes a direct link to its location on Google Maps.

For Users:

Secure Authentication: A complete user registration and login system using secure, token-based authentication (JWT).

Profile Management: A dedicated profile page where users can view and update their personal information and change their password.

For Property Owners:

Protected Dashboard: A private, secure dashboard for owners to manage their listings.

Full CRUD Functionality: Owners can Create, Read, Update, and Delete their properties.

Multi-Image Uploads: An advanced form that supports uploading multiple images with a live preview and delete functionality.

💻 Technology Stack

This project is a full-stack application built with modern and robust technologies.

Backend:

Language: Java 17+

Framework: Spring Boot 3

Security: Spring Security with JWT (JSON Web Tokens)

Database: PostgreSQL

API: RESTful API architecture

Frontend:

Core: HTML5, CSS3, Vanilla JavaScript (ES6+)

Styling: Tailwind CSS for a modern, utility-first design.

🚀 Getting Started

To run this project on your local machine, follow these steps:

1. Backend Setup:

Navigate to the backend folder.

Update the src/main/resources/application.properties file with your PostgreSQL database credentials.

Open the project in your favorite IDE (like Eclipse or IntelliJ) and run the RentooAppApplication.java file.

The server will start on port 8081.

2. Frontend Setup:

Navigate to the frontend folder.

Open the index.html file in your web browser.

You can now use the full application locally.
